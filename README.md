# Sirah Analytics Intelligence Platform

AI-driven Governance for Enterprise Web Analytics — by Sirah Digital.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

```sh
# Clone the repository
git clone https://github.com/riyazlive04/web-analytics-agent.git

# Navigate to the project directory
cd web-analytics-agent

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Pages

- `/` — Main architecture dashboard
- `/demo-flow` — Cinematic AI workflow simulation
- `/agents-flow` — Interactive agent architecture explorer

## Deployment

Build for production:

```sh
npm run build
```

Output is in the `dist/` directory.
